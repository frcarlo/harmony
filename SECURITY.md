# Security Policy

## Supported Versions

Currently, only the latest release of HArmony is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 3.x.x   | :white_check_mark: |
| < 3.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within HArmony, please do not disclose it publicly.

Instead, please report the vulnerability privately by opening a [Security Advisory](https://github.com/frcarlo/harmony/security/advisories/new) on GitHub or by sending a direct message to the maintainers.

We will endeavor to respond to your report within 48 hours and work with you to resolve the issue as quickly as possible. We will also coordinate the public disclosure of the vulnerability after a patch has been released.
